# Holidaze — Accommodation Booking App

A modern front-end accommodation booking application built for Noroff Project Exam 2.

## Description

Holidaze allows visitors to browse venues, customers to make bookings, and venue managers to manage their properties — all powered by the Noroff API v2.

## Tech Stack

- React 18
- Vite
- TypeScript
- Tailwind CSS
- React Router v6
- Noroff API v2

## Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation

```bash
git clone https://github.com/lacdart2/holidaze-pe2.git
cd holidaze-pe2
npm install
npm run dev
```

## Environment Variables

No environment variables required. API key is generated at runtime.

## Features

- Browse and search venues
- View venue details and availability calendar
- Register as customer or venue manager
- Customer: create bookings, view upcoming bookings, update avatar
- Venue Manager: create, edit, delete venues, view bookings
